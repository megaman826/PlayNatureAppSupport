# [한국어]
# 앱 개요
자연이랑놀자는 아이들이 동물, 곤충, 물고기 등 자연 관련 영상을 안전하고 간편하게 시청할 수 있도록 설계된 앱입니다. 단순하고 직관적인 인터페이스를 제공하며, 모든 콘텐츠는 앱에 포함되어 있어 인터넷 연결 없이 이용할 수 있습니다.
# 앱 사용 방법
1. 앱을 실행하면 가로 화면으로 시작됩니다.
2. 카테고리(동물, 곤충, 물고기 등)를 선택합니다.
3. 원하는 영상을 터치하면 바로 재생됩니다.
4. 모든 콘텐츠는 화면 전환 없이 이용할 수 있습니다.
# 주요 기능
- 자연 관련 영상 콘텐츠 제공
- 가로 화면 전용 인터페이스
- 어린이 친화적 단순 UI
- 카테고리 기반 콘텐츠 탐색
- 모든 콘텐츠 오프라인 사용 가능
# 사용 권한
앱은 서비스 제공을 위해 민감한 기기 권한을 요구하지 않습니다. 모든 콘텐츠는 앱 내부에 포함되어 있으며 추가 다운로드가 필요하지 않습니다.

이 정보는 개인을 식별할 수 없으며 광고나 마케팅에 사용되지 않습니다.

# [English]
# App Overview
"자연이랑놀자" helps children watch nature-related videos such as animals, insects, and fish in a safe and simple environment. The app uses an intuitive interface, and all content is built into the app for full offline use.
# How to Use
1. Launch the app; it opens in landscape mode.
2. Select a category (animals, insects, fish, etc.).
3. Tap a video to start playback instantly.
4. All content is accessed within a single screen without page transitions.
# Key Features
- Nature-related video content
- Landscape-only interface
- Child-friendly simple UI
- Category-based browsing
- Full offline access to all content
# Permissions Used
The app does not require sensitive device permissions. All content is included in the app, and no additional downloads are needed.
# Data Collection & Privacy
The app uses Firebase Analytics to improve service quality and app stability. The following anonymous usage data may be collected:

Possible data collected:
- App usage duration and frequency
- Feature usage statistics
- Device model and OS version
- Non-identifiable device information

This data does not identify users personally and is not used for advertising or marketing.
